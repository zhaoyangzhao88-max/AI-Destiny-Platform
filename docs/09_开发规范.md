## 9.4 Git 工作流

> 【待确认】—— 建议从 `E:\VSCODE` 父仓库中分离独立仓库

**当前问题：** 本项目目录 `E:\VSCODE\AI-Destiny-Platform` 属于 `E:\VSCODE` 大仓库的一部分。未来可能造成的问题：
- 与其他项目（Pixelle-Video、iptv-engine 等）共享 git 历史
- commit 信息混杂
- 独立发布 / CI 困难

**建议方案：**
1. 初始化独立 git 仓库
2. 或使用 `git submodule` 方式管理
3. 或保持现状并使用 `.gitignore` 隔离（最不推荐）
