# 黄金数据集交叉验证报告

> **验证目标:** 1990-08-15 12:00 的日柱/时柱/日主正确值
> **验证日期:** 2026-07-17
> **原始占位值 (10_测试规范.md §10.4):** 日柱=戊寅, 时柱=戊午, 日主=戊
> **待验证值 (lunar_python 计算结果):** 日柱=壬子, 时柱=丙午, 日主=壬

## 验证方法

### 方法 1: cnlunar 独立库计算

**来源:** `cnlunar` v0.2.4 — 纯 Python 农历/八字计算库，与 lunar_python 无代码共享，独立实现

**结果:**

```
1990-08-15 12:00:
  year8Char:   庚午
  month8Char:  甲申
  day8Char:    壬子   ← 与 lunar_python 一致
  twohour8Char: 丙午  ← 与 lunar_python 一致
```

**结论:** ✅ 与 lunar_python 完全一致，互相印证。

---

### 方法 2: 手工推导（基准日 + 天数差）

**基准日:** 1990-01-01，日柱 = **丙寅**（经多个万年历网站确认）
**推导过程:**

```
步 1: 计算天数差
  1990-01-01 → 1990-08-15 = 226 天

步 2: 确定基准日索引
  丙寅 = 六十甲子第 3 位 (0-based index = 2)

步 3: 推算目标日索引
  target_index = (2 + 226) % 60 = 48

步 4: 查六十甲子
  索引 48 → 天干 48 % 10 = 8 → 壬
           地支 48 % 12 = 0 → 子
  → 日柱 = 壬子
```

**验证:** 壬子 = 六十甲子第 49 位 (0-based index = 48)，核对无误。

**结论:** ✅ 手工推导与 lunar_python 一致。

---

### 方法 3: 时柱手工推导（五鼠遁）

**日干:** 壬
**五鼠遁口诀:** "丁壬庚子居" → 壬日子的子时天干为庚

```
时辰推算:
  子时 (23-01): 庚子
  丑时 (01-03): 辛丑
  寅时 (03-05): 壬寅
  卯时 (05-07): 癸卯
  辰时 (07-09): 甲辰
  巳时 (09-11): 乙巳
  午时 (11-13): 丙午 ← 12:00 在午时
```

**结论:** ✅ 时柱 = 丙午，与 lunar_python 和 cnlunar 一致。

**来源:** 多个八字排盘网站搜索结果
